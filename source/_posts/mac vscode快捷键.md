* 多处光标 `option + 鼠标左键`
* 向下插入一行 `command + enter`
* 向上插入一行 `shift + command + enter`
<!-- more -->
* 向上 / 下 移动行 `option + up / down`
* 向上 / 下 复制行 `shift + option + up / down`
* 查找 `command + F`
* 查找下一个 `command + G`
* 选取全部与当前匹配的 `shift + command + L`
* 注释 / 解注释 `command + /`
